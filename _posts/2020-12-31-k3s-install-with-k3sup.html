--- layout: post title: HA K3S install with k3sup date: 2020-12-31 18:12:04.000000000 +02:00 type: post published: true status: publish categories: - GNU/Linux - Tutorial tags: - GNU/Linux - k3s meta: author: endikap100 --- Deploy a HA K3S cluster with etcd using k3sup <

For the first master node execute:

<k3sup install --ip --user --cluster --ssh-key --k3s-version --k3s-extra-args '--disable=servicelb'

<

For the following master nodes execute:

<k3sup join --ip --user --server-user --server-ip --server --ssh-key --k3s-version --k3s-extra-args '--disable=servicelb'

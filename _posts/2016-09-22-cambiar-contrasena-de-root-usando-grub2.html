--- layout: post title: Cambiar contraseña de root usando grub2 date: 2016-09-22 15:57:15.000000000 +02:00 type: post published: true status: publish categories: - GNU/Linux - Seguridad - Tutorial tags: - grub2 meta: author: endikap100 ---

  • Presionar la tecla e en el menú de grub2
  • Buscar la linea que empiece por linux, y se le añade en el final "rw init=/bin/bash" como aparece en la foto de el principio.
  • Pulsamos Ctrl+x y tendremos una shell como root
  • Para cambiar la pasword usaremos el comando # passwd
  • Ya tenemos nueva contraseña para el usuario root, para reiniciar: # /sbin/reboot -f

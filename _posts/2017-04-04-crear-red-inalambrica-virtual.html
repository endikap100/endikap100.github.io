Para crear red inalámbrica virtual ejecutamos el siguiente comando:

iw dev wlan0 interface add wlan0_ap type managed addr 12:34:56:78:ab:ce

En este ejemplo la interface wlano es desde la que se quiere generar la interface virtual, la interface wlan0_ap el nombre que le damos a la interface y 12:34:56:78:ab:ce la mac que le asignamos.

Aunque uno no sea muy partidario de google, aveces se necesitan los servicios de google play para algunas aplicaciones...

Para este ejemplo se va a usar una una maquina virtual android 4.4.4 montada con genymotion.
Lo primero es descargar las google apps desde la wiki de cyanogenmod,  descargar las google apps para versiones de cyanogenmod equivalentes a el android donde se quieran instalar (ejemplo para android 4.4.4 se descargaran las google apps de cyanogenmod 11

Se necesitaran:

GoogleLoginService.apk que esta dentro del archibo comprimido Core/gsflogin-all.tar.xz
GoogleServicesFramework.apk que esta dentro del archibo comprimido Core/gsfcore-all.tar.xz

Phonesky.apk que esta dentro de el archivo comprimido Core/vending-all.tar.xz (para versiones anteriores a la creo que este archivo se llama Vending.apk)

Una vez tenemos los apk que vamos a instala tambien necesitaremos la el android sdk (esta despues de toda la mierda de android studio)

Una vez descomprimidas entraremos en la carpeta platform-tools desde la terminal y ejecutaremos las siguientes ordenes

Para montar la particion donde copiaremos los apk-s

adb shell mount -o remount,rw -t yaffs2 /dev/block/mtdblock0 /system

Para añadirle permisos de escritura

# Allow writing to app directory on system partition
adb shell chmod 777 /system/app

y para copiar los las apk-s

# Install following apk
adb push GoogleLoginService.apk /system/app/.
adb push GoogleServicesFramework.apk /system/app/.
adb push Phonesky.apk /system/app/. # Vending.apk in older versions
